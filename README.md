# 🔍 VeriFact AI – Automatic Fact Checker

## 📌 Overview

*VeriFact AI* is an AI-powered automatic fact-checking system designed to detect misinformation and verify claims in real time. It analyzes input text, extracts factual claims, and compares them against a verified database to provide accurate verdicts with confidence scores.

Automated fact-checking systems typically follow a pipeline of claim extraction, evidence retrieval, and verification to determine truthfulness ([SourcePulse][1]).

---

## 🚀 Features

* ✅ *Real-Time Fact Checking*
  Instantly verifies user-input claims.

* 🧠 *AI-Based Claim Extraction*
  Identifies key factual statements from text.

* 📊 *Verdict Classification*

  * True
  * Likely False
  * Unverified

* 📈 *Confidence Scoring*
  Displays probability of correctness (e.g., 99%).

* 🗂️ *Facts Database*
  Stores verified facts with explanations and sources.

* ⚙️ *Batch Processing*
  Analyze multiple posts with concurrency control.

* 📊 *Dashboard Analytics*
  Tracks throughput, processing time, and verdict breakdown.

---

## 🏗️ System Workflow

1. *Input Content* – User enters text/post
2. *AI Extraction* – System identifies factual claims
3. *Matching Engine* – Compares with verified database
4. *Verdict Generation* – Assigns truth label + confidence
5. *Explanation Output* – Provides reasoning and source

---

## 🖥️ Modules

* *Fact Checker* – Verify individual claims

* *Facts Database* – Manage verified facts
